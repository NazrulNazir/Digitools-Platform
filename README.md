# 🚀 Digitools Platform

## 📖 Description
Digitools Platform is a modern and responsive web application built from a Figma design. This project allows users to explore digital tools, view pricing plans, and interact with the platform through a clean and user-friendly interface. The project focuses on reusable components, responsive layout, and smooth user interaction.

---

## 🛠️ Technologies Used
- React.js
- Tailwind CSS
- React-Toastify
- JavaScript (ES6)
- HTML5
- CSS3

---

## ✨ Features
1. 📱 Fully Responsive Design  
   - The website is optimized for mobile, tablet, and desktop devices using Tailwind CSS.

2. 🔔 Interactive Notifications  
   - Implemented toast notifications using React-Toastify for better user feedback.

3. 🧩 Reusable Components  
   - Built reusable React components to make the project scalable and maintainable.
